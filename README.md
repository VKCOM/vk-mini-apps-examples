# mini-app-shop-boilerplate
мини-апп онлайн магазин

## Запуск

1. Клонирование репозитория
   ```bash
   git clone https://github.com/pasha-nikitin-2003/mini-app-shop-boilerplate.git
   ```
2. Установка зависимостей
   ```bash
   npm install
   ```
3. Запуск проекта
   ```bash
   npm start
   ```
